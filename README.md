# Fetch Me ✉️

A Slack App for developers to make API requests using a slash command 💻

---

[![serverless](http://public.serverless.com/badges/v3.svg)](http://www.serverless.com)

## Install

    $ git clone https://github.com/cfahlgren1/fetchme
    $ cd fetchme
    $ npm install
